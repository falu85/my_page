Gym Tracks

Objective:

  • Create a workout application where you can select exercises by searching from an existing list or create new ones
  • Add these selected exercises to your current workout along with the completed sets and repitions

Why We Care:

  • This area was a great interest for the majority of the team
  • In the history of the class, all groups who attempted this workout application did not perfom well. We took it as a challenge to be the group to break this pattern
  • At the time there was no suitable application for users to have a positive experience

The Process:

  • Performed extensive information gathering from interviews of potential application users
  • Collaborated on general User Interface (UI) design and functionality
  • Iterated through many UI designs and made changes based on TA and peer feedback.
  • Designed an implemented the exercise search and input pages using Blazor and CSS, designed application logo and button selections using Canva and hand-drawn art respectively
  • Designed a page to track your workout history, with alerts for new personal bests, as well as a graphical representation of each workout to track improvements

Results:

  • Our group was the first group to break the pattern and obtained a B+ on this project
  • Ended up also including a tracker of user workouts and alerts for new personal bests!
