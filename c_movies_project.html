C-Movies: Movie Lovers Unite!

Background:

  • There used to be a site called CalgaryMovies.com that would have a listing of all the theatres in and around Calgary, their movies, and showtimes.
    • Unfortunately, the site cosed down some time ago, and as a movie lover myself, I would like to recreate this site!

Objective:

  • To fan the flame of Calgary movie lovers and make a site where they can find all the movies that are playing in and around the city in this open source project
  • Each movie listing includes the poster of the movie, length, runtime, synopsis, show times, and theatre they are playing at

The Process:

  • As a proof of concept, scraped The Landmark Cinemas website for current movies using Selenium Webdriver (Python)
  • Using the scraped movie titles, employed The Movie Database API to get the following information: genre, original movie language, and movie rating. This is all done using Python.
  • All movie information is placed in a JSON object for ease of access

Results so far:

  • I am currently learning Next.js to create a a web application to display this information

Goal:

  • My goal is for this to become a fully functioning website within the next few months
